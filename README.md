## 🌀 How autopoe Works

autopoe combines several AI assistants into one workspace. Each agent has a role, like writing text, generating code, or analyzing data. The agents talk to one another using AI “chats” inside autopoe.

You use a single user interface to:

- Create new tasks your AI team will handle
- Assign roles to agents
- Watch agent conversations and decisions
- Save results or export data

The system helps avoid confusion by keeping tasks and messages organized automatically. You get AI help without managing the details.
